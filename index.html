Welcome to the project page of Dynamake

What is Dynamake about?

Dynamake is an attempt to envision, design, and implement an interactive code extrinsic programming environment. Further, this environment is to require little configuration effort before getting the hands dirty.

What is Dynamake not about?

There is no aspiration of developing a programming environment for solving any kind of problem. Instead, the environment is developed in an open-ended, experimental, and intuitive manner with a focus on the fundamentals.

Why was this project started?

Most popular programming environments seem to dichotomize programs and processes simply by centering around code - even most object oriented software.
